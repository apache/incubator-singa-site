A Distributed Deep Learning Library

Get Started
Apache SINGA is an Apache Top Level Project, focusing on distributed training of deep learning and machine learning models

Scalablility

SINGA parallelizes the training and optimizes the communication cost to improve training scalability.

Efficiency

SINGA builds a computational graph to optimizes the training speed and memory footprint.

Usability

SINGA has a simple software stack and Python interface to improve usability.
